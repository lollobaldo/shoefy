What does Shoefy do?

1. Concept

Shoefy is a sport-equipment fetching robot, specifically designed to assist with ice skates booking and distribution at ice rinks. It comes with an interface for reservation making and storage traking and an admin panel where the robot's path and environment are set. The system also includes a disinfecting mechanism and a foot measurer which can be used by users that do not know the correct size of their feet. Shoefy is really flexible when it comes to the environment so the product could be suitable for other simillar alleys such as bowling ones.

However, Shoefy's most notable asset is the fact that it removes the need of an on-desk employee and reduces the human contact which is extremely important today. What's more, Shoefy's booking and storage system will almost completely reduce the overcrowding around the rinks as the ice skating enthusiasts will be able to constantly keep track of the availablitlity.

1.2. Functionality

It comes with two main parts: a fetching robotand a booking application. The two subsystems are specifically designed to cohesively work together and have clear functionalities:

  • An application which creates bookings, shows availability of equipment at any given time and produces a unique booking QR code for each successful reservation.
  • An already built robot with control system which when given a QR code moves tothe location of the item specified in the booking and brings it to the customers. The robot also comes with a foot measuring device and a cleaning sprayer fordisinfection. An admin panel is provided for easy setup of the environment and designing the path around the place.

Robot

The robot is the main component of the system. Its purpose is to fetch the skates for the user and collect their shoes, which will be placed on the shelf. Whenthe user returns with the same QR code, the robot will fetch the shoes, and then return the skates.It has three main functionalities: navigation, lifting, and pulling. The robot is equiped with several distance sensors and infrared sensors which help it navigate around by following lines on the floor while avoiding any potential obstacles. The lines(path) must be set up before the first start of the robot. This makes it able to be used in rooms of any size and shape. The robot is able to extend to a height of 2 meters with a payload of up to 5kg.
